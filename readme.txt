== Description ==

Clockwork for WP is a plugin to work with the browser extension [Clockwork](https://github.com/itsgoingd/clockwork-chrome).   Clockwork is a Chrome extension for PHP development, extending Developer Tools with a new panel providing all kinds of information useful for debugging and profiling your PHP applications, including information about request, headers, get and post data, cookies, session data, database queries, routes, visualisation of application runtime and more.

== Installation ==

Using this plugin is very simple. All you have to do is:

1. Upload the `clockwork-wp` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Enable debug mode in WordPress wp-config.php file
